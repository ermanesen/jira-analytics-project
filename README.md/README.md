# Jira-Based Analytics Workflow Project

This project simulates an end-to-end analytics workflow integrating Business Analyst and Data Analyst responsibilities using Python, SQL, and DuckDB.

## Technologies
- Python (Pandas, Matplotlib, Seaborn)
- SQL (DuckDB)
- Jupyter Notebook
- Git / GitHub

## Workflow
1. Business requirement definition
2. Data cleaning and transformation
3. SQL-based analytics workflow
4. KPI calculation and visualization
